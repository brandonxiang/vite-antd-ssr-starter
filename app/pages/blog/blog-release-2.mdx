# How to use the blog mode

## Quick Start

Please take a look at [mdx](https://mdxjs.com/) and [react-router](https://reactrouter.com/), and learn how to use them.

First, clone the project and install the dependencies.

```bash
npx degit brandonxiang/vite-antd-ssr-starter
npm install
```

Second, please use mdx for markdown writing in the app/pages/blog directory.

Third, deploy to github pages.


## Blog Settings

Please go to app/pages/blog/index.tsx, config `BLOG_CONFIG` to define your blog list.

## Router Settings

Please take a look at router/menus.tsx, use [react-router](https://reactrouter.com/) to define your routes.
